Careful Paws — Front-End Portfolio Project

Welcome to Careful Paws, a pet boarding and daycare website built as a portfolio project to showcase my front-end development skills. This project demonstrates my ability to create responsive, visually appealing websites with attention to detail, layout, and accessibility —skills I’m excited to bring to a team.

Why I Built This

I created this site to demonstrate:
- layout techniques
- Responsive design that works across desktop, tablet, and mobile
- Integration of images and visual storytelling
- A warm, user-focused design that’s easy to navigate
- My ability to learn and implement new front-end concepts quickly

This project is designed as a beginner-friendly portfolio site, showing practical front-end work with HTML and CSS, aligned with real-world requirements for a small business website.


Technologies Used

- HTML5
- CSS3
- Responsive design principles
- GitHub-hosted images
